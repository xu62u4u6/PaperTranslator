### 使用教學
#### Windows版
  
1. 貼上論文內容
2. 按下Enter
3. 按下Ctrl + Z 鍵
4. 按下Enter
5. 享受沒有空格的翻譯內容
  
#### Linux版
1. 貼上論文內容
2. 按下Enter
3. 按下Ctrl + D 鍵
4. 享受沒有空格的翻譯內容
   
#### Online版本
1. 執行上面一格 (%%capture那格)
2. 開始使用 PaperTranslator !!
